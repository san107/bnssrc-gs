# doori 

## 요약 

- R : 읽기. 
- RW : 읽기/쓰기. 
- SET : 1로쓰기. 
- RST : 0쓰기. 
- PULSE : 누를땐 1 떼면 0 
- 각 어드레스단위는 비트 및 워드단위임. 

## 차단막 어드레스맵(BIT) 

- 차단막 상태
  - M1B 차단막 상승완료상태(R) : 008220 
  - M1D 차단막 하강완료상태(R) : 008222 
- 차단막 작동 
  - M101 차단막 하강버튼(R/W) : 008354 (누를때 1 떼면 0) 
  - M106 차단막 상승버튼(R/W) : 008359 (누를때 1 떼면 0) 
- 현장 차단막 버튼의 운전모드 셀렉터 상태( 운전모드 원격상태, 자동상태시에만 원격프로그램에서 작동가능) 
  - M300 운전모드 자동(R) : 008673 
  - M301 운전모드 수동(R) : 008674 
  - M302 운전모드 원격(R) : 008675 
  - M303 운전모드 현장(R) : 008676 
- 태풍모드/일반모드 
  - M1300 일반모드/태풍모드(R/W) : SET AND RST : 010273
